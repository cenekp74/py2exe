py2exe.com - the first online python compiler

py2exe.com is a free online tool that compiles Python projects to executables in the cloud. It provides the simplest way to convert python files to exe from both windows or linux environment. Whether you're a developer looking to share your Python application without requiring end-users to install Python or simply aiming for seamless distribution, py2exe.com simplifies the process.

This site is currently still in development. If you encounter any errors or if you have any idea on how to make it better, please email me at admin@py2exe.com. The source code is publicly available on github. I will add a proper readme and comments to the code soon.
